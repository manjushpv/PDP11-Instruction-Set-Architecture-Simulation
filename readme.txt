	Simulation file execution instruction

----------------------------------------------------------------------------------------

--> Open terminal on Mac or CommandPrompt on Windows
--> Enter "python path"
	path refers to location on .py(code file) you copied from our code package
	Ex: python /Users/x_laptop/Desktop/code.py
--> It prompts for input --> Input ascii file path
	Enter path of ascii file stored
--> It prompts to Choose Debug Mode: ON-1 OFF-0
	Choose accordingly

--> Once you return both inputs above the code executes and simulation results are ON.
